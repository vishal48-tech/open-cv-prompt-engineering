
## Run Locally

Clone the project

```bash
  git clone https://github.com/vishal48-tech/open-cv-prompt-engineering.git
```

Go to the project directory

```bash
  cd open-cv-prompt-engineering
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Run the script

```bash
  py <file_name.py>
```

## About Project

In this repo there are currently 4 projects.
- First: Blur person, keep background clear.
- Second: Nose point detection.
- Third: Posture detection.
- Fourth: American Sign Language detection.
